Types Of List in HTML

1. Unordered List

2. Ordered List

3. Defination List

Unordered List

Unordered List is kind of list which is not in proper order, like its just in the bulleted form or say any like that, no use of number or counting.

The format of unordered list can be changed using "type="circle"" form or like that.

Ordered List

Ordered List is a kind of list which is in a order form or say its a countable list and it has numerics or alphabets for counting in it.

The format of ordered list can be changed using "type="1"" form or like that.

Defination List

Defination List is being used in here for that like of dictonary thing like name and a separate and meaning below it. This list is used to give a term and a meaning below it.
